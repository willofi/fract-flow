# 🔗 인터랙션 모델

MindMap AI는 촉각적 정밀함을 최우선으로 생각합니다. 모든 상호작용은 인지 부하를 최소화하면서도 제어 능력을 최대화할 수 있도록 설계되었습니다.

---

## 📐 50/25/본체 분할 모델

**이동**, **크기 조절**, **연결** 간의 충돌을 해결하기 위해 노드의 인터랙션 영역을 정밀한 기하학적 영역으로 분할했습니다:

### 1. 연결 영역 (중앙 50%)
- **위치**: 상, 하, 좌, 우 각 모서리의 중앙 50% 구간.
- **트리거**: 마우스 근접 시 `handle-indicator-circle`이 나타납니다.
- **동작**: 이 원형 핸들을 클릭하거나 드래그하면 노드 연결이 시작됩니다.

### 2. 크기 조절 영역 (측면 25% + 코너)
- **위치**: 각 가장자리의 바깥쪽 25% 구간과 네 개의 모서리.
- **동작**: `NodeResizer`를 사용하여 노드의 크기를 동적으로 조절할 수 있습니다.

### 3. 본체 내부 (편집 및 이동)
- **위치**: 노드 내부의 전체 표면.
- **이동**: 클릭 시 노드가 선택되며, 드래그 시 이동합니다.
- **편집**: 더블 클릭 시 **실시간 코드 에디터**가 활성화됩니다. `prismjs` 기반의 구문 하이라이팅을 통해 마크다운과 코드를 실제 렌더링 결과와 거의 동일한 색상으로 편집할 수 있습니다.

## 🛠 현대적 내비게이션 UI

사용자가 캔버스의 넓은 공간을 확보할 수 있도록 불필요한 장식을 제거한 **베젤리스(Bezelless)** 내비게이션을 지향합니다.

### 좌측 상단 패널
- **ArrowLeft 버튼**: 이전 페이지로 돌아가는 직관적인 화살표 아이콘. 호버 시 왼쪽으로 살짝 이동하는 마이크로 애니메이션이 적용되어 방향성을 암시합니다.
- **제목 편집기**: 캔버스 위에 바로 노출된 미니멀한 입력창으로, 테두리 없이 텍스트 본연의 가독성을 높였습니다.

### 우측 상단 툴바
- **컴팩트 툴바**: 뷰 리셋, 노드 추가, 공유, 수동 저장 버튼을 하나의 그룹으로 묶었습니다.
- **미니멀 디바이더**: 버튼 간의 구분선을 절반 높이(`h-2`)로 줄여 시각적 노이즈를 최소화했습니다.

## 🎯 픽셀 단위의 제로 갭(Zero-Gap) 연결

노드 기반 UI에서 흔히 발생하는 시각적 문제 중 하나는 연결선이 노드 테두리에서 1-2픽셀 앞에서 멈추는 "플로팅 엣지(Floating Edge)" 현상입니다. 우리는 **초정밀 제로 차원 핸들(Pinpoint Zero-Dimension Handles)**을 통해 이를 해결하여 끊김 없는 건축학적 디자인을 완성했습니다.

## 🎨 다크 모드 맞춤형 색상

노드 우클릭 시 나타나는 색상 팔레트는 사용자의 현재 테마를 감지합니다.
- **라이트 모드**: 배경색을 투과하는 파스텔 톤.
- **다크 모드**: 흰색 텍스트의 가시성을 위해 명도와 채도가 지능적으로 조정된 어두운 톤.
