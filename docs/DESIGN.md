# 💅 디자인 시스템

MindMap AI의 디자인은 **"콘텐츠 우선(Content Over Chrome)"** 원칙에 따라 설계되었습니다. 사용자의 아이디어가 시각적으로 강조될 수 있도록 최소한의 절제된 환경을 지향합니다.

---

## 🖤 단색 슬레이트(Slate) 미학

일반적인 도구에서 흔히 볼 수 있는 파란색 강조색을 배제했습니다. OKLCH 표준을 기반으로 한 단색의 **슬레이트/그레이(Slate/Grey)** 팔레트를 사용하여 전문적인 청사진(Blueprint) 같은 느낌을 제공합니다.

### 컬러 토큰
- **Primary**: 순수한 검정색의 거친 느낌 없이 높은 대비를 제공하는 풍부한 슬레이트(`oklch(0.3 0.02 260)`).
- **Background**: 테마에 따라 깨끗한 오프화이트(Off-white) 또는 깊은 네이비(Navy)를 사용하여 장시간 작업 시 눈의 피로를 줄입니다.
- **Node Colors**: 무한 캔버스를 압도하지 않으면서 카테고리를 구분할 수 있도록 고안된 고투명 파스텔 톤 세트.

---

## 🌓 지능형 다크 모드

MindMap AI는 `next-themes`를 사용한 고품질 다크 모드를 제공합니다.

- **자동 적응**: 단순한 UI 패널뿐만 아니라 **캔버스 그리드, 노드 테두리, 연결선**까지 색상이 동적으로 전환됩니다.
- **미니맵 글래스모피즘(Glassmorphism)**: 미니맵은 `backdrop-blur` 효과를 사용하여 캔버스의 하부 구조를 보여주는 동시에 가독성을 유지합니다.

---

## 📐 마이크로 타이포그래피

마인드맵 특성상 작은 노드가 많이 포함될 수 있으므로 **마이크로 폰트 크기(Micro-FontSize)** 로직을 구현했습니다:
- **기본 크기**: 8px.
- **두께(Weight)**: 작은 크기에서도 높은 가독성을 제공하는 Semibold.
- **마크다운 지원**: `@tailwindcss/typography`를 통합하여 작은 공간 내에서도 계층적 콘텐츠(제목, 굵게, 리스트 등)를 표현할 수 있습니다.

## 🛠 CSS 엔지니어링

성능과 일관성을 유지하기 위해 **Tailwind CSS v4**의 최신 기능을 활용했습니다:
- **@theme inline**: 즉각적인 전역 업데이트를 위한 중앙 집중식 컬러 변수.
- **글래스모피즘**: 패널에 `backdrop-blur-xl`을 광범위하게 사용하여 깊이감 있는 SaaS 미학을 구현.
- **트랜지션 피지컬(Transition Physics)**: 색상 전환부터 노드 크기 조절까지 모든 움직임에 `cubic-bezier(0.4, 0, 0.2, 1)`를 적용하여 프리미엄 수준의 부드러운 사용감을 제공합니다.
